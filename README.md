# BootstrapVite

## Instructions for use

1. Install dependencies:

```
npm install
```

2. Run vite server:

```
npm run dev
```

3. Open browser on <http://localhost:8080>

## Instructions for homework

1. First Exercise

- Header con un titulo
- nav con 3 enlaces
- main

- footer con derechos reservados

2. Second Exercise

- Cambiar el color del fondo del body
- Centrar el titulo
- Agregar padding al main

3. Third Exercise
